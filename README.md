### 插件说明
 + 默认上班时间为 9:00
 + 默认下班时间为 18:00
 + 默认周末类型为 双休

 + 引入了moment js日期框架
 + 优化底层计算日期
 + 增加 自定义上下班时间
 + 增加 周末距离类型
